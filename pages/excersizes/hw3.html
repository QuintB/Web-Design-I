These are important tags that I need to know if I want to be a successful web designer!

HTML stands for: Hyper Text Markup Language

What does this bracket mean <tagname>: Open Tag

What does this bracket mean </tagname>: Close Tag

<html> declaration at the beginning of the html document

<p> indicates a new paragraph or a paragraph style

<body> starting the body of the page, which is where all the contant the user sees is

<aside> denotes content other than the content the tag is found in

<br> break; makes a new line

<div> tells the html which tag in CSS to use

<footer> start of the footer

<link> denotes a new link to another website or file

<li> an item in an unordered list

<nav> defines a set of navigation links

<object> usually at the start of an embedded item, like a video or pdf

<section> stuff like chapters, headers, footers, or any other sections of a document

<span> groups objects together; does nothing by itself

<style> tells the site which text or css style to use

<title> the title of the page that the browser uses

<a> hyperlink

<article> independend, self contained content

<blockquote> specifies a section that is quorted from another source

<img> image tag

<h1> to <h6> heading styles

<strong> bold

<table> start of an html table

<td> table cell tag

<tr> table row tag

<ul> unordered list, anything with an image in front of the items instead of enumeration

What is a tag: a way to affect only one part of the website
